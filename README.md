# 📣 Master-The-Auction 💰

---

## 1️⃣ Project Overview (프로젝트 개요)

- **📝 프로젝트 이름**: **Master - The - Auction (MTA)**  
- **📖 프로젝트 설명**:   실시간 경매 사이트입니다.
  - **판매자가 되어 경매물품 등록 가능**  
  - **입찰에 참여하여 물건 낙찰받기 가능**
  - **기존 입찰가보다 높은 입찰가로 입찰 시, 기존 입찰한 사람에게 포인트 돌려주는 기능**
  - **각 판매자 및 게시판에 좋아요, 싫어요 기능**  
  - **경매물품 후기를 공유**할 수 있는 커뮤니티 기능 제공
